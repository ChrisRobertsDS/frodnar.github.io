About Chris

I'm an intellectually curious researcher who loves to dig into and analyze all kinds of R&D and business data – especially in Python. I have experience in intellectual property, chemistry, and math, and I am happiest learning about and applying new programming skills particularly in environments that make space for innovation and change.

Projects

Experimenting with coauthorship networks in Python

I wanted to make a proof-of-concept network visualization, so I once again turned to Dimensions.ai COVID-19 publication data extracted on Google BigQuery. I analyzed coauthorship for the 200 most-cited publications using the Python package metaknowledge and visualized the network with Gephi. The SQL query involved some fun RegEx in order to trick metaknowledge into thinking these were Scopus results. Check out the full dataset here.
